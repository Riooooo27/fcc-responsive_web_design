Introduction to Python

Python is a popular and versatile high-level programming language known for its simplicity, readability, and elegance. It was created by Guido van Rossum and first released in 1991. Since then, it has gained tremendous popularity among developers, making it one of the most widely used languages in the world.

One of Python's primary strengths is its focus on code readability, achieved through the use of a clean and straightforward syntax. This readability allows programmers to express concepts and ideas more concisely and clearly, enhancing collaboration and reducing the time required to write, debug, and maintain code.

Another advantage of Python is its extensive standard library, which provides a vast collection of modules and functions for various purposes, such as handling files, network communication, web development, data manipulation, and much more. This rich set of tools simplifies development, as programmers can leverage existing functionalities instead of building everything from scratch.

You can also do a lot of things using python, like:

  • Artificial Intelligence (AI) and Machine Learning (ML)
  • Data Science
  • Scripting and Automation
  • Scientific Computing
  • Big Data and Data Engineering

Overall, Python's simplicity, readability, versatility, and strong community support have contributed to its widespread adoption, making it an excellent choice for both beginners and experienced developers for a broad range of projects and applications.

Variables

In Python, variables are used to store data values. They act as containers that hold different types of information, such as numbers, text, or objects. Variables allow you to give names to values, making it easier to reference and manipulate them throughout your code.

To create a variable in Python, you need to use the assignment operator (=). The variable name is placed on the left side of the = sign, and the value you want to store in the variable is placed on the right side. For example:

age = 30
name = "John Doe"

Variable names in Python can consist of letters (both uppercase and lowercase), numbers, and underscores. However, they must follow some rules:

  • The name cannot start with a number.
  • Variable names are case-sensitive (e.g., age and Age are different variables).
  • Avoid using reserved keywords as variable names (e.g., if, while, for, etc.).

You can change the value of a variable after it has been assigned and As Python is dynamically typed, you can assign a different type of value to the same variable:

x = 5
print(x) # Output: 5

x = 10
print(x) # Output: 10

x = "Hello"
print(x) # Output: Hello

Python also allows you to assign multiple variables in a single line using a technique called "multiple assignment":

a, b, c = 1, 2, 3
Variable Scope

In most programming languages, including Python, variables have different scopes, and the scope is determined by where the variable is defined. The two primary types of variable scope are:

  • Global Scope: A variable declared in the global scope is accessible throughout the entire program, including all functions and blocks. It is defined outside any function or block and is available from the point of declaration until the end of the program. Global variables are typically defined at the beginning of the code, outside any functions.
  • Local Scope: A variable declared in the local scope is only accessible within the specific function or block in which it is defined. Once the function or block finishes executing, the variable's scope ends, and the variable is destroyed. Local variables are defined inside functions, loops, or conditional blocks.

Here's a simple example to illustrate variable scope in Python:

# Global variable
global_var = 10

def example_function():
  # Local variable
  local_var = 5
  print("Inside the function:", local_var, global_var)
 
example_function()
print("Outside the function:", global_var)

# Output:
# Inside the function: 5 10
# Outside the function: 10

In this example, global_var is a global variable because it is defined outside any function. It can be accessed both inside and outside the example_function.

On the other hand, local_var is a local variable because it is defined inside the example_function. It can only be accessed within that function and is not available outside of it.

Data types

In Python, data types define the type of data that can be stored in a variable. Python is a dynamically-typed language, which means you don't need to specify the data type explicitly when declaring a variable. Instead, Python will automatically assign the appropriate data type based on the value you assign to the variable.

Here are some common data types in Python:

  • int : Represents a whole number (-1, 2, 3, 4, etc)
  • float : Represents floating-point numbers (3.14, -0.5, 2.0, etc)
  • complex : Represents complex numbers (1+2j, -3k, etc)
  • str : Represents a strings of characters ('Hello', "Love you", etc)
  • bool : Represents boolean values, which can be either True or False.
  • list : Represents an ordered collection of elements, enclosed in square brackets '[]' ([1, 3, 9])
  • tuple : Similar to lists, but enclosed in parentheses (()), and they are immutable (cannot be changed after creation), e.g., (1, 2, 3).
  • dict : Represents a collection of key-value pairs, enclosed in curly braces "{}", e.g., {"name": "John", "age": 30}.
  • set : Represents an unordered collection of unique elements, enclosed in curly braces "{}", e.g., {1, 2, 3}.
  • None : Represents the absence of a value or a null value. It is often used to indicate the absence of data or as a default value.
if-else Statement

In Python, the if-else statement is used for conditional execution. It allows you to execute a block of code based on whether a specific condition is true or false. The basic syntax of the if-else statement is as follows:

if condition:
  # Code block executed if the condition is True
else:
  # Code block executed if the condition is False

You can also have multiple conditions using elif (short for "else if") statements. The elif statement is used when you have multiple conditions to check, and it comes between the if and else statements. It works like a chain of conditions, and as soon as one condition is true, the corresponding block of code is executed, and the rest of the chain is skipped.

Example:

x = 7

if x > 10:
  print("x is greater than 10")
elif x > 5:
  print("x is greater than 5 but not greater than 10")
else:
  print("x is 5 or less")

In this example, the second condition x > 5 is true, so the second print statement will be executed, and the else block will be skipped.

Loops

In Python, loops are used to repeat a block of code multiple times. Python provides two types of loops: for loop and while loop.

  • For Loop

    • The for loop is used to iterate over a sequence (such as a list, tuple, string, etc.) and execute a block of code for each element in the sequence. Syntax:

    for element in sequence:
      # Code block to be executed for each element

    Example:

    numbers = [1, 2, 3, 4, 5]
    for num in numbers:
      print(num)     Output: 1
    2
    3
    4
    5
  • While Loop

    • The while loop is used to repeatedly execute a block of code as long as a specified condition is true. Syntax:

    while condition:
      # Code block to be executed while the condition is true

    Example:

    count = 0
    while count < 5:
      print(count)
      count += 1

    Output:

    0
    1
    2
    3
    4

    It's essential to ensure that the condition in a while loop eventually becomes false; otherwise, the loop will run indefinitely, leading to an infinite loop.

Function

In Python, a function is a reusable block of code that performs a specific task. Functions are used to break down complex problems into smaller, more manageable pieces, making code more organized, easier to read, and less prone to errors.

A function in Python is defined using the 'def' keyword, followed by the function name, parentheses '()', and a colon ':'. Any parameters (inputs) that the function requires are specified inside the parentheses.

Here's the basic syntax of a function in Python:

def function_name(parameter1, parameter2, ...):
  # Code block of the function
  # Perform some operations
  return result

Let's break down the components:

  • Function Definition : The def keyword indicates the start of a function definition.
  • Function Name : The name of the function is user-defined and should follow the same rules as variable names.
  • Parameters (optional) : These are variables that act as placeholders for the values that will be passed into the function when it's called. Functions can have zero or more parameters, separated by commas.
  • Code Block : The indented block of code under the function definition represents the body of the function. It contains the instructions or operations the function performs.
  • Return Statement (Optional) : The return keyword is used to specify the value that the function will return as its result. If no return statement is used, the function will implicitly return None.

Example of a simple function:

def greet(name):
  return 'Hello, ' + name

message = greet('Alice')
print(message)

Output:

Hello, Alice

In this example, we defined a function named greet that takes one parameter (name). When the function is called with the argument 'Alice', it returns the greeting 'Hello, Alice'. The result is stored in the variable message, and we print it to the console.

Functions are fundamental building blocks of Python programming. They allow you to modularize your code, promote reusability, and improve the overall organization and structure of your programs.
